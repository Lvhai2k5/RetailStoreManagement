# 🛒 Retail Store Management System

## 📌 Giới thiệu
Retail Store Management System là hệ thống quản lý cửa hàng bán lẻ, hỗ trợ quản lý hàng hóa, đơn hàng và theo dõi hoạt động kinh doanh trong cửa hàng.

Hệ thống giúp:
- Quản lý sản phẩm (thêm, sửa, xóa)
- Quản lý đơn hàng
- Theo dõi tồn kho

---

## 🎯 Mục tiêu dự án
- Xây dựng hệ thống quản lý cửa hàng bán lẻ
- Áp dụng mô hình MVC
- Thực hành quy trình Agile với Jira
- Quản lý source code bằng GitHub
